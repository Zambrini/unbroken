SIMULATION BOUNDARY // READ BEFORE ENTRY

The fight is real. The global service is not—yet.

Version 0.5.0 is a local vertical slice. Its audience, queue, challenger identities, victory record, and between-round evolution are deterministic presentation. There is no live multiplayer, persisted history, account system, trained model, or frame-by-frame AI control in this checkpoint.
